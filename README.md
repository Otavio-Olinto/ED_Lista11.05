# ED_Lista11.05
## Verificação de Conformidade

Este desafio consiste em criar uma aplicação, em Java, que controle uma fila de um brinquedo de um parque de diversões.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A fila deve ser importada de uma biblioteca própria(disponível em: https://github.com/Otavio-Olinto/ED_Lista09.03);
3. A fila deve ser composta por 30 pessoas com o nomes: Pessoa1, Pessoa2..;
4. A aplicação deve avaliar os requsitos mínimos para andar no brinquedo.

Idade>= 17 anos
Altura>= 1,61m
```

```
----------------------
Cliente
----------------------
1. nome: String
2. idade: int
3. altura: Double
----------------------

Preencher os parâmetros de idade e altura com métodos aleatórios.

Margem da idade: 10~40
Margem de Altura: 1,35~2,00
```
